Peer dependencies: @angular/core, @angular/common, rxjs

Configuration Options

Property Type Required Description
clientId string Required Your UAE Pass application client ID
clientSecret string Optional Client secret (for confidential clients)
redirectUri string Required Redirect URI after authentication
isProduction boolean Optional Use production environment (default: false)
scope string Optional OAuth scope (default: profile:general)
language UaePassLanguageCode Optional UI language (En/Ar, default: En)
storage UaePassStorageMode Optional Token storage (Session/Local/None)
tokenProxyUrl string Optional Proxy URL for token exchange
userInfoProxyUrl string Optional Proxy URL for user info requests

API Reference

UaePassAuthService

Signals (Reactive State)

  • status(): Signal<UaePassAuthStatus> - Current authentication status
  • tokens(): Signal<UaePassTokens|null> - Access and refresh tokens
  • profile(): Signal<UaePassUserProfile|null> - User profile information
  • error(): Signal<string|null> - Last error message
  • isAuthenticated(): Signal<boolean> - Computed authentication state

Methods

  • redirectToAuthorization(): Promise<void> - Redirect to UAE Pass login
  • handleRedirectCallback(url?: string): Promise<void> - Handle callback
  • logout(): void - Logout and clear session
  • resetError(): void - Clear error state

Quick Start Example

1. Configure in app.config.ts

import { provideUaePass, UaePassLanguageCode, UaePassStorageMode } from 'sensei-uaepass'; export const appConfig: ApplicationConfig = { providers: [ provideHttpClient(withFetch()), provideUaePass({ clientId: 'your-client-id', redirectUri: 'http://localhost:4200/uae-pass/callback', isProduction: false, language: UaePassLanguageCode.En, storage: UaePassStorageMode.Session, tokenProxyUrl: '/api/uae-pass/token', userInfoProxyUrl: '/api/uae-pass/userinfo', }), ], };

2. Add Routes

import { UaePassCallbackComponent } from 'sensei-uaepass'; export const routes: Routes = [ { path: 'uae-pass/callback', component: UaePassCallbackComponent }, // ... other routes ];

3. Use in Component

import { UaePassAuthService, UaePassAuthStatus } from 'sensei-uaepass'; @Component({ template: ` @if (auth.status() === Status.Authenticated) { <h2>Welcome, {{ auth.profile()?.fullnameEN }}!</h2> <button (click)="logout()">Logout</button> } @else { <uae-pass-login-button (pressed)="login()"></uae-pass-login-button> } ` }) export class HomeComponent { private readonly auth = inject(UaePassAuthService); readonly Status = UaePassAuthStatus; async login(): Promise<void> { await this.auth.redirectToAuthorization(); } logout(): void { this.auth.logout(); } }

Proxy Server

The proxy server is highly recommended for security:

  • Client Secret Protection - Keeps secrets secure server-side
  • CORS Handling - Avoids cross-origin issues
  • Token Security - Prevents token exposure in browser

You can implement the proxy with any backend technology:

  • Node.js/Express
  • ASP.NET Core
  • Python/FastAPI
  • Java/Spring Boot
  • PHP/Laravel
